4. 개인정보 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 아래의 경우는 예외입니다.

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 따라 수사기관 등이 요청하는 경우

6. 이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 열람 권리: 본인의 개인정보 처리 현황 확인 요청
  • 정정 권리: 오류가 있는 개인정보 수정 요청
  • 삭제 권리: 불필요한 개인정보 삭제 요청 (법령상 보존 의무 항목 제외)
  • 처리 정지 권리: 개인정보 처리 정지 요청

권리 행사는 고객지원 이메일(vingly.official@gmail.com)로 요청하시면 10일 이내 조치합니다.

7. 쿠키 사용

회사는 로그인 상태 유지, 서비스 이용 분석을 위해 쿠키를 사용합니다. 브라우저 설정에서 쿠키를 비활성화할 수 있으나, 일부 서비스 이용에 제한이 있을 수 있습니다.

8. 개인정보 보호 조치
  • 비밀번호 암호화 저장 (단방향 암호화)
  • 개인정보 접근 권한 최소화 (담당자 외 접근 불가)
  • 보안 접속(HTTPS) 적용
  • 정기적인 보안 점검
